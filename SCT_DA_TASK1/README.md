Task 1 – Excel Sales Dashboard
📌 Description

This project focuses on cleaning, analyzing, and visualizing the Superstore Sales Dataset using Microsoft Excel. The dataset was prepared by handling inconsistencies and organizing the data for analysis. Interactive dashboards were created using Pivot Tables, Pivot Charts, Slicers, and Conditional Formatting to present key business insights.

The dashboard provides an overview of Total Sales, Profit by Category, and Sales Trends over Time, enabling users to quickly identify high-performing products, profitable categories, and overall sales performance. It also highlights top and low performers through conditional formatting, making the dashboard more informative and user-friendly.

🎯 Objective
Clean and prepare the sales dataset.
Create Pivot Tables for data summarization.
Build interactive charts and dashboards.
Visualize Total Sales, Profit by Category, and Sales Trends.
Highlight high and low performers using Conditional Formatting.
